# General Configuration

General settings control the core behavior of your ThetaSecure instance including branding, locale, and system-wide defaults.

**Navigation:** Sidebar → Configuration → General

## Settings

| Setting | Description |
|---------|-------------|
| **Organization Name** | Your organization's display name |
| **Login Page Branding** | Custom logo, colors, and welcome message |
| **Default Language** | System language preference |
| **Time Zone** | Default timezone for logs and reports |
| **Session Defaults** | Default session timeout values |
| **Support Contact** | Help desk contact information shown to users |
