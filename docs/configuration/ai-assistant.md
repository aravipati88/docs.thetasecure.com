# AI Assistant

ThetaSecure includes an AI-powered assistant that helps administrators with common tasks, troubleshooting, and security recommendations.

**Navigation:** Sidebar → Configuration → AI Assistant

## Features

- **Natural language queries** — Ask questions about your security posture
- **Guided troubleshooting** — Step-by-step help for common issues
- **Security recommendations** — AI-driven suggestions for improving your identity security score
- **Policy generation** — Help creating access and authentication policies

## Configuration

| Setting | Description |
|---------|-------------|
| **Enable/Disable** | Toggle the AI Assistant on or off |
| **Data Scope** | Configure what data the assistant can access |
| **Response Language** | Preferred language for responses |
