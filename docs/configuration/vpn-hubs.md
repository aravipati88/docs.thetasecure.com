# VPN Hubs

VPN Hubs are the gateway endpoints through which remote users connect to your network via ThetaSecure's VPN.

**Navigation:** Sidebar → Configuration → VPN Hubs

## Hub Management

| Field | Description |
|-------|-------------|
| **Hub Name** | Friendly name for the endpoint |
| **Region** | Geographic location |
| **Status** | Online, Offline, Degraded |
| **Connected Users** | Current active connections |
| **Capacity** | Maximum concurrent connections |

## Adding a VPN Hub

1. Click **+ Add Hub**
2. Configure the hub name and region
3. Set capacity limits
4. Configure routing rules
5. Click **Save**
