# Security Configuration

System-wide security settings that apply across your ThetaSecure instance.

**Navigation:** Sidebar → Configuration → Security

## Settings

| Setting | Description |
|---------|-------------|
| **Password Policy** | Global minimum password requirements |
| **Account Lockout** | Failed attempt threshold and lockout duration |
| **Session Security** | Cookie settings, CSRF protection, concurrent session limits |
| **IP Allowlist/Blocklist** | Global IP restrictions for admin access |
| **Device Trust** | Auto-approval rules for device registration |
| **CORS Settings** | Allowed origins for API cross-origin requests |
