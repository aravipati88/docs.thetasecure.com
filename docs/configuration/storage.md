# Storage

Configure storage backends for audit logs, session recordings, and other persistent data.

**Navigation:** Sidebar → Configuration → Storage

## Storage Types

| Type | Used For |
|------|----------|
| **Log Storage** | Audit logs and event data |
| **Recording Storage** | RDP/SSH session recordings |
| **Backup Storage** | System configuration backups |

## Retention Settings

Configure data retention policies to meet compliance requirements:
- Default: 90 days
- Extended: 365 days
- Compliance mode: Immutable storage for regulatory requirements
