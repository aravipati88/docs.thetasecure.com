# Notifications

Configure how ThetaSecure sends notifications to users and administrators.

**Navigation:** Sidebar → Configuration → Notifications

## Notification Types

| Type | Description |
|------|-------------|
| **Email** | SMTP configuration for sending emails |
| **Admin Alerts** | Notifications for security events and system issues |
| **User Notifications** | Password reset, MFA enrollment, account activation emails |
| **Webhook** | HTTP callbacks for external integrations |

## Email (SMTP) Configuration

Configure your outbound email server for sending notifications:
- SMTP server address and port
- Authentication credentials
- TLS/SSL settings
- Sender email address and display name
