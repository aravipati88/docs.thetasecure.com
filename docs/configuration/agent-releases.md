# Agent Releases

Manage the distribution and versioning of ThetaSecure agent software deployed to end-user devices and network infrastructure.

**Navigation:** Sidebar → Configuration → Agent Releases

## Agent Types

| Agent | Platform | Purpose |
|-------|----------|---------|
| **ZTNA Agent** | Windows, macOS, Linux | Zero Trust Network Access client |
| **VPN Client** | Windows, macOS, Linux, iOS, Android | VPN connectivity |
| **Directory Connector** | Windows Server | Active Directory synchronization |

## Release Management

- View available agent versions
- Set the recommended version for each platform
- Force upgrade policies for outdated agents
- Download agent installers for distribution
