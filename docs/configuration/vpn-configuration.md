# VPN Configuration

Configure ThetaSecure's VPN capabilities for providing secure remote access to your network resources.

**Navigation:** Sidebar → Configuration → VPN Configuration

## Settings

| Setting | Description |
|---------|-------------|
| **VPN Protocol** | Supported VPN protocols |
| **Tunnel Mode** | Split tunnel vs full tunnel |
| **DNS Settings** | DNS servers for VPN clients |
| **IP Pool** | Client IP address range |
| **Encryption** | Cipher and key exchange settings |
| **Keep-Alive** | Connection persistence settings |
