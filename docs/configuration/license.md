# License

View and manage your ThetaSecure license details, usage, and subscription status.

**Navigation:** Sidebar → Configuration → License

## License Information

| Field | Description |
|-------|-------------|
| **License Type** | Edition (Standard, Professional, Enterprise) |
| **Status** | Active, Expiring, Expired |
| **Expiry Date** | License expiration date |
| **Licensed Users** | Maximum users allowed |
| **Current Usage** | Number of active users |
| **Features** | Enabled feature modules |

## License Portal

Access the full license management portal at [license.thetasecure.com](https://license.thetasecure.com) for renewals, upgrades, and detailed usage analytics.
