# VPN Statistics

Aggregated analytics for VPN usage, performance, and capacity planning.

**Navigation:** Sidebar → Audit → VPN Statistics

## Metrics

| Metric | Description |
|--------|-------------|
| **Active Sessions** | Current VPN connections |
| **Peak Connections** | Maximum concurrent connections |
| **Total Data Transferred** | Bandwidth consumption |
| **Average Session Duration** | Mean connection time |
| **Top Users** | Most active VPN users |
| **Top Destinations** | Most accessed internal resources |
