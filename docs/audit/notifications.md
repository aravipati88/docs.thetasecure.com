# Notification Audit Logs

Track all notifications sent by ThetaSecure — emails, admin alerts, and webhook deliveries.

**Navigation:** Sidebar → Audit → Notifications

## Tracked Events

| Event | Description |
|-------|-------------|
| **Email Sent** | Notification email delivered |
| **Email Failed** | Email delivery failure |
| **Alert Triggered** | Security or system alert generated |
| **Webhook Delivered** | Webhook callback completed |
