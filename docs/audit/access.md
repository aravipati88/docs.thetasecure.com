# Access Audit Logs

Track resource access events and access policy evaluations.

**Navigation:** Sidebar → Audit → Access

## Tracked Events

| Event | Description |
|-------|-------------|
| **App Access** | User accessed a web or network application |
| **Policy Evaluation** | Access policy matched and action taken |
| **Server Session** | RDP/SSH session initiated or terminated |
| **Token Issued** | OAuth/OIDC token granted |
| **RADIUS Auth** | RADIUS authentication request processed |
