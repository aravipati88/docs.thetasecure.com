# System Audit Logs

Track administrative and system-level changes to your ThetaSecure configuration.

**Navigation:** Sidebar → Audit → System

## Tracked Events

| Event | Description |
|-------|-------------|
| **Configuration Change** | Any system setting modified |
| **Policy Created/Updated** | Access or authentication policy changes |
| **Application Configured** | App registration or modification |
| **License Updated** | License key or subscription changes |
| **Connector Modified** | Directory sync connector changes |
| **Agent Updated** | Agent release or distribution changes |
