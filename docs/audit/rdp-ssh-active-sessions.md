# RDP/SSH Active Sessions

Real-time view of all active RDP and SSH sessions through ThetaSecure's remote access gateway.

**Navigation:** Sidebar → Audit → RDP/SSH Active Sessions

## Session Details

| Field | Description |
|-------|-------------|
| **User** | Connected user |
| **Server** | Target server hostname/IP |
| **Protocol** | RDP or SSH |
| **Started** | Session start time |
| **Duration** | Elapsed time |
| **Source IP** | User's IP address |

## Administrative Actions

| Action | Description |
|--------|-------------|
| **Monitor** | View the live session (shadow mode) |
| **Terminate** | Force-disconnect the session |
