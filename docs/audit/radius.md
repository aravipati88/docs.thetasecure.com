# RADIUS Audit Logs

Detailed logs of all RADIUS authentication requests processed by ThetaSecure.

**Navigation:** Sidebar → Audit → RADIUS

## Log Details

| Field | Description |
|-------|-------------|
| **Timestamp** | When the request was processed |
| **Client** | RADIUS client device name/IP |
| **Username** | Authenticating user |
| **Result** | Accept, Reject, Challenge |
| **Reason** | Failure reason (if rejected) |
| **NAS IP** | Network Access Server IP |
