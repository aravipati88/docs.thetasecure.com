# RDP/SSH Recordings

Playback recorded RDP and SSH sessions for audit, compliance, and incident investigation.

**Navigation:** Sidebar → Audit → RDP/SSH Recordings

## Recording List

| Field | Description |
|-------|-------------|
| **User** | Who initiated the session |
| **Server** | Target server |
| **Protocol** | RDP or SSH |
| **Date** | Session date and time |
| **Duration** | Total session length |
| **Size** | Recording file size |

## Playback

Click on any recording to play it back in the browser. The player supports:
- Play/pause, seek, and speed controls
- Full-screen viewing
- Command search (SSH sessions)
- Keystroke logging (if enabled)

:::tip Compliance
Enable session recording for all privileged access sessions. This provides a complete audit trail for compliance with SOC 2, ISO 27001, and similar standards.
:::
