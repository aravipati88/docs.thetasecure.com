# VPN Flow Logs

Detailed network flow records for all traffic passing through ThetaSecure's VPN infrastructure.

**Navigation:** Sidebar → Audit → VPN Flow Logs

## Log Details

| Field | Description |
|-------|-------------|
| **Timestamp** | Flow start time |
| **User** | Connected user |
| **Source IP** | Client IP address |
| **Destination** | Target resource IP and port |
| **Protocol** | TCP/UDP |
| **Bytes** | Data transferred |
| **Duration** | Connection duration |
| **Action** | Allowed or Blocked |
